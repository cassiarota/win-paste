PasteNowWin

Windows 上的剪贴板历史管理器,参考 macOS 的 PasteNow 设计。系统自带的 Win+V 重启即清空, PasteNowWin 用本地数据库持久化保存剪贴板历史,通过全局快捷键随时调出、搜索、粘贴。

⬇ 下载安装版 (Setup.exe) 便携版 (zip)

Windows 10 / 11 · 自带运行时,无需安装 .NET · 当前版本 最新

核心功能

● 剪贴板历史

自动保存文本 / 图片 / 文件,跨重启持久化,自动去重。

● 全局快捷键

一键弹出历史框搜索粘贴,可在设置里自定义组合键。

● 纯文本粘贴

一键把最近一条以无格式纯文本粘贴到当前窗口。

● 类型分组与搜索

全部 / 文本 / 图片 / 文件 / 置顶 标签筛选,顶部实时搜索。

● 排除规则

按进程名排除(如密码管理器),敏感复制不被记录。

● 过期自动清理

可设 1 / 7 / 30 / 90 天或永不,置顶项始终保留。

● 多屏跟随光标

弹窗在光标所在显示器弹出,适配 per-monitor 高 DPI。

● 提示音

可选复制时播放提示音,操作有反馈。

● 自动检查更新

启动检查新版本,有更新托盘提示,一键回到本页下载。

默认快捷键
Ctrl Shift V 打开剪贴板历史框
Ctrl Shift B 将最近一条以纯文本粘贴

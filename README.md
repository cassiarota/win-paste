# PasteNowWin

一个 Windows 上的**剪贴板历史管理器**,灵感来自 macOS 的 [PasteNow](https://pastenow.app)。

Windows 自带的剪贴板(`Win+V`)**一重启就清空**,用起来也不顺手。PasteNowWin 把你复制过的
**文本 / 图片 / 文件**都本地持久化保存下来,随时用一个全局快捷键调出、搜索、选中、粘贴。

> 平台:Windows 10 / 11(64 位)。程序常驻系统托盘,**自带运行时,无需安装 .NET**。

---

## 下载与安装

到 [**Releases**](https://github.com/cassiarota/win-paste/releases) 下载,任选一种:

| 文件 | 适合 | 用法 |
|---|---|---|
| `PasteNowWin-*-Setup.exe` | 大多数人 | 双击安装(**无需管理员、无 UAC**,装到当前用户目录),可勾选开机自启 |
| `PasteNowWin-*-win-x64.zip` | 想免安装 / 放 U 盘 | 解压后双击 `PasteNowWin.exe` 直接运行 |

也可以从**官网**直接下载最新版:在 Cassian Engines 门户首页点 **PasteNowWin** 卡片进入,页面内即有下载按钮。

启动后**没有主窗口**——程序在系统托盘(屏幕右下角,橙色星形图标)。右键托盘图标可打开菜单。

## 快速上手

| 快捷键(默认) | 作用 |
|---|---|
| **Ctrl + Shift + V** | 弹出剪贴板历史框,搜索并选择要粘贴的内容 |
| **Ctrl + Shift + B** | 把**最近一条**历史以**纯文本**(去格式)粘贴到当前窗口 |

在历史弹窗里:

- `↑` / `↓` 选择,`Enter` 粘贴,`Ctrl + Enter` 以纯文本粘贴,`Esc` 关闭
- 顶部输入框可**实时搜索**;上方 `全部 / 文本 / 图片 / 文件 / 置顶` 标签可按类型**筛选**
- **右键**任意一条:粘贴 / 纯文本粘贴 / 置顶 / 删除
- 弹窗会出现在**鼠标所在的显示器**上;点击别处自动关闭

> 说明:Windows 的 `Win+V` 已被系统占用,所以默认沿用 PasteNow 的 `Ctrl+Shift+V` / `Ctrl+Shift+B`。
> 两个快捷键都**可以自己改**(见下)。

---

## 功能一览

- 自动保存**文本 / 图片 / 文件**三类内容,**跨重启持久化**,相同内容自动去重并置顶
- **搜索** + **类型分组**,快速找到历史条目
- 粘贴过的条目自动排到最前(最近使用优先)
- **置顶**常用条目,常驻列表顶部、不被自动清理
- **自定义快捷键**:两个全局快捷键都可录制成你喜欢的组合
- **提示音**:可选「复制时播放提示音」
- **多显示器 + 高 DPI**:弹窗跟随光标所在屏幕,缩放自适应
- **过期自动清理**:可设 1 / 7 / 30 / 90 天或永不(置顶项不受影响)
- **排除规则**:指定某些程序(如密码管理器),它们的复制不被记录
- **自动检查更新**:启动时检查新版本,有更新时托盘气泡提示,一键打开下载

## 设置

右键托盘图标 →「设置…」。

- **开机自启动** — 登录 Windows 时自动启动
- **复制时播放提示音** — 复制成功时播放一声提示音
- **历史过期时间** — 选择 1 / 7 / 30 / 90 天或永不过期;超时的非置顶记录会被自动清理
- **排除规则** — 每行填一个进程名(如 `KeePass`、`1Password`,可不带 `.exe`),来自这些程序的复制不被记录
- **自定义快捷键** — 见下
- **清空历史** — 可选保留置顶项,或全部清空

### 自定义快捷键

1. 打开「设置」,找到**快捷键**一栏。
2. 点击「打开剪贴板历史」或「纯文本粘贴最近一条」对应的按钮,它会变成「按下快捷键…」。
3. 直接按下你想要的组合键(**需要包含 `Ctrl` / `Shift` / `Alt` / `Win` 至少一个修饰键**)。
4. 松开即自动保存并立即生效;按 `Esc` 可取消。
5. 如果该组合键已被其它程序占用、或与另一个快捷键冲突,会弹窗提示你换一个,原快捷键保持不变。

## 数据位置与卸载

- 历史数据库:`%LOCALAPPDATA%\PasteNowWin\`(SQLite)。删除该目录即清空全部历史与设置。
- 卸载:安装版从「设置 → 应用」或控制面板卸载;便携版直接删除解压出来的文件夹。卸载不会自动删数据目录,可手动删除上面的路径。

---

## 注意事项

- 粘贴注入**无法作用于以管理员权限运行的窗口**(本程序以普通权限运行)。如确需向管理员窗口粘贴,
  需要让 PasteNowWin 也以管理员身份运行。
- 自定义的全局快捷键是**系统级**的,会覆盖个别程序内同样组合键的功能;如有冲突,换一个组合即可。

## 从源码构建(开发者)

需要 [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)(Windows)。

```powershell
# 开发运行
dotnet run --project src/PasteNowWin/PasteNowWin.csproj

# 发布为自带运行时的单文件 exe
dotnet publish src/PasteNowWin/PasteNowWin.csproj -c Release -r win-x64 `
  --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```

技术栈:C# / .NET 8 / WPF,使用 Win32(`AddClipboardFormatListener`、`RegisterHotKey`、
`SendInput`)实现剪贴板监听、全局热键与粘贴注入,历史存于 SQLite。
